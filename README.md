# pirate peggle
Thema = piraten

# beschijving
Je schiet een kannons kogel/of pistool kogel 
die kogel kaatst op stenen/houten tonnen naar de overkant/onderkant
je kan een upgrade krijgen als je een bepaalde ton raakt/de upgrade is dat je een blunderbus schiet met meerde kogels.je moet zoveel mogelijk barrels stik maken en de bal naar de overkant krijgen ook beweegt de 

# speler 
je bent een piraat met een wapen (kannon of pistool ligt aan level)
je bent in een gevecht op land of zee en je moet op de vijand schieten.

# gameplay
je krijgt score door de barrels teraken maar je krijgt nog meer als je de vijand raakt je krijgt bonuses door of heelveel punten tehalen of door upgrades teraken.
je wint bij een bepaalde score/ of als je de vijand zovaak hebt geraakt.

# structuur 
Links of boven zit de speler ligt aan level.
midden zitten de targets
rechts of onder zit de vijand
